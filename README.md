# 熱管理技術研究中心（TMRC）網站原始碼

本資料為中央大學熱管理中心（Thermal Management Research Center, TMRC）網站之原始碼檔案。

## 網站預覽
https://tmrc-ncu.github.io/

## GitHub Repository
https://github.com/tmrc-ncu/tmrc-ncu.github.io

## 網站說明
本網站為靜態網站（Static Website），主要由 HTML、CSS 與圖片檔案組成，無需資料庫或後端程式即可運行。

## 檔案結構

- `index.html`：網站首頁
- `css/`：網站樣式檔案（CSS）
- `images/`：網站圖片與圖示
- `members/`：教師與團隊介紹頁面
- `research/`：研究領域介紹頁面
- `news/`：最新消息頁面

## 使用方式
開啟 `index.html` 即可進入網站首頁，其餘頁面皆由首頁連結。

## 備註
若需修改網站內容，可直接編輯對應的 HTML 檔案與圖片資源後重新部署。
