# 熱管理中心網站原始碼

本資料為中央大學熱管理中心（TMRC）網站之原始碼。

## 網站預覽
https://tmrc-ncu.github.io/

## GitHub Repository
https://github.com/tmrc-ncu/tmrc-ncu.github.io

## 檔案結構

index.html  
網站首頁

css/  
網站樣式（CSS）

images/  
網站圖片檔案

members/  
教師與團隊介紹頁面

research/  
研究領域介紹頁面

news/  
最新消息頁面

## 使用方式

開啟 `index.html` 即可進入網站首頁，其餘頁面皆由首頁連結。
